Experiment X - Subject/Topic

Description

This experiment is intended to generate a large variety of unique loglines for films using pre-existing characters and plot hooks, solely for entertainment purposes.

Technical

This experiment was coded in javascript. A series of lists were created that were placed in a template string in a precise order, and within each list were a number of different

character names, settings, plot hooks, and descriptive words that would be randomly selected each time the string is parsed. This javascript code was then hooked up to a basic HTML

window such that each click of a button would generate a new film logline and display it on the screen.
